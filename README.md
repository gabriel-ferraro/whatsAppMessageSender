<h2>WhatsApp Message Sender app </h2>

<p>Ao digitar o número desejado, não esquecer do prefixo estadual; o prefixo 55 não é necessário. <br>exemplo de numero válido: 4191234567899</p>

<p>Para enviar uma mensagem a um número sem ter a necessidade de adicioná-lo como contato, é possível com uma intent encaminhar o conteúdo para uma URL oferecida pela API do whatsApp. É necessário que o aparelho possua o aplicativo WhatsApp instalado. </p>
